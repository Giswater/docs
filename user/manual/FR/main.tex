{\Large \textbf{Préambule}} 
\newline
Le manuel d’utilisation de Giswater 3 a pour objectif de répondre aux questions relatives ŕ l’installation et ŕ l’utilisation du programme Giswater. Le présent document couvre toutes les informations nécessaires pour les utilisateurs ou intéressés pour savoir comment installer, configurer et utiliser Giswater 3. 
Le software a été licencié sous une licence GNU-GLP 3 (https://www.gnu.org/licenses/gpl-3.0).
L’index donne des informations sur la liste des éléments disponibles et est construit en trois bloques principaux qui sont : installation, utilisation et annexes variables.
