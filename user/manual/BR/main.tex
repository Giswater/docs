{\Large \textbf{Preâmbulo}} 
\newline
O Manual de Usuário do Giswater 3 tem o objetivo de responder as perguntas relativas a instalaçăo e execuçăo do programa Giswater. O presente documento cobre toda a informaçăo necessária para que os usuários e partes interessadas saibam como instalar, configurar e operar com o Giswater 3.

O software é licenciado pela licença GNU-GLP 3 (https://www.gnu.org/licenses/gpl-3.0).

O índice fornece informaçőes sobre a lista de elementos disponíveis e estăo estruturadas em tręs blocos principais que săo: instalaçăo, uso e vários anexos.



\newline
